Euclidean Rhythm Music Sequencer Using Actors in Python

Daniel P. Prince

Abstract
A real-time sequencer that implements the Euclidean rhythm algorithm is created for creative generation of drum sequences by musicians or producers. Modern synchronization techniques using the Actor model are used to simplify the communication required for interactivity and musical timing. Other techniques such as generators and higher order functions are also used to simplify the program. The resulting application sends Musical Instrument Digital Interface (MIDI) data interactively to another application for sound generation.
